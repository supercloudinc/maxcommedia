Welcome to Sparky Framework!

Sparky Framework from HotJoomlaTemplates.com is free generic Joomla template licensed under GNU/GPL that can be used as a starting point for development of Joomla templates. Sparky's main adventages are flexibility (your site can have literaly any layout and dimensions) and ease of use (create any layout using only drag'n'drop).

Please use the tabs and subtabs to navigate through the options. On each tab, you will find some text that describe the options of this tab. Also, if you move pointer over the option name, a popup with detailed explanation will be displayed.

Enjoy your creations with Sparky!
