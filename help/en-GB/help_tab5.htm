Menu Options

All menus from your Joomla site that's listed in menu manager are also listed here. For each menu you can select a menu type. After that, appropriate option for this menu type will be shown. There are 5 menu types available.

Drop-down menu:
This is most commonly used menu type. It contains horizontal buttons. Submenu items will be shown in the drop-down box.

Horizontal menu:
In this menu type the submenu items will be shown horizontally under the main menu items.

Mega menu:
More complex menu system that allows you to put several levels of menu items as well as modules into the submenu. Please read documentation on our site.

Vertical (accordion):
The main menu items are stacked vertically while the submenu items are shown under the main menu item if user clicks on it.

Joomla classic:
This will show a classic Joomla menu. This menu type is recommended if you don't have submenu items.
